CSS CƠ BẢN >> Border

Thuộc tính border-style

Mô tả kiểu đường viền sẽ được hiển thị. Các giá trị:

  • none
  • solid
  • dotted
  • dashed
  • double
  • groove
  • ridge
  • inset
  • outset

Thuộc tính border-width

Mô tả độ dày của đường viền. Các giá trị:

  • thin
  • medium
  • thick
  • số pixel cụ thể

Thuộc tính border-color

Mô tả màu sắc của đường viền. Các giá trị:

  • tên màu: red
  • giá trị rgb: rgb(255,0,0)
  • chỉ số hex: #ff0000

Ví dụ

  • Khai báo thuộc tính cho từng cạnh:
    border-top-style: solid;
    border-left-style: dotted;
    border-right-style: dashed;
    border-bottom-style: double;
  • Khai báo nhiều thuộc tính cùng lúc:
    border: ridge medium lightblue;
